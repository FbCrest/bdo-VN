# Black Desert Online Việt hóa
**Hướng dẫn cài đặt**

**※ Việt hóa**
- Giải nén file languagedata_en.loc vào thư mục Black Desert Online\ads.

- tạo thư mục prestringtable\font trong Black Desert Online.
